    </div>
    <div style={{padding: '10px 0', marginBottom: '15px'}}>

      <span style={{color: '#606060', fontWeight: 600, fontSize: '16px', marginLeft: '20px'}}><b>$color--&lt;variable-name&gt;</b>, где <b>&lt;variable-name&gt;</b> — имя переменной</span>

    <div style={{padding: '10px 0', marginBottom: '15px'}}>
      <span style={{color: '#111111', fontWeight: 700, fontSize: '20px'}}>Все значения цветов пишутся в HEX-формате строчными буквами </span>

    </div>
    <div style={{padding: '10px 0', marginBottom: '15px'}}>
      <span style={{color: '#111111', fontWeight: 700, fontSize: '20px'}}>Переменные с цветами можно разделять пустой строкой, если это имеет логический смысл</span>
      <div style={{color: '#111111', fontWeight: 700, margin: '20px 0', padding: '0 30px'}}>Пример:</div>
      <div style={{backgroundColor: '#dedede', borderRadius: '4px', padding: '30px', margin: '15px 30px'}}>
        $color--primary: #002f6c; <span style={{color: '#909090', marginLeft: '40px'}}>// Эти цвета относятся к основным (просто разные оттенки)</span><br/>
        $color--primary-accent: #0041a0; <span style={{color: '#909090', marginLeft: '40px'}}>// Эти цвета относятся к основным (просто разные оттенки)</span><br/>
        $color--primary-dark: #001e46; <span style={{color: '#909090', marginLeft: '40px'}}>// Эти цвета относятся к основным (просто разные оттенки)</span><br/><br/>

        $color--success: #00a314; <span style={{color: '#909090', marginLeft: '40px'}}>// Это другая группа основных цветов</span><br/><br/>

        $color--font-secondary: #909090; <span style={{color: '#909090', marginLeft: '40px'}}>// Это цвета для шрифтов</span><br/>
        $color--font-muted: #bcbcbc; <span style={{color: '#909090', marginLeft: '40px'}}>// Это цвета для шрифтов</span><br/>
      </div>

      <div style={{color: '#111111', fontWeight: 700, margin: '20px 0', padding: '0 30px'}}>Пример:</div>

      <div style={{padding: '10px 0', marginBottom: '15px'}}>

        </div>
        <div style={{marginTop: '10px'}}>
          Где <b>&lt;class-name&gt;</b> — имя класса, который будет создан, а <b>&lt;variable-name&gt;</b> — имя переменной, на основе которой будет создан класс.
        </div>
      </span>
      </div>
